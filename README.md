# ooke

A lightweight CMS and web application framework built on toke.

**toke on ooke** — build light, ship fast.

## What is ooke?

ooke is a content management system and web application framework written entirely in toke. It compiles to a single native binary with no runtime dependencies. Pages render to static HTML at build time; dynamic routes execute as compiled native code.

## Status

Phase 1 in development. See [toke/docs/progress.md](../toke/docs/progress.md) for story tracking.
